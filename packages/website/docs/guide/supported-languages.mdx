## Unknown Languages

When you use a language that isn't in the list above, a generic code icon `</>` will be displayed:

<LanguageTabs groupId="unknown-languages">
  <LanguageTab language="javascript">
    ```js
    // Known language - has specific icon
    console.log("JavaScript has an icon");
    ```
  </LanguageTab>
  <LanguageTab language="cobol">
    ```cobol
    * Unknown language - generic icon
    DISPLAY "COBOL is not supported yet".
    ```
  </LanguageTab>
</LanguageTabs>
